## Sobre

Criação de um formulário de cadastro simples e reponsivo com a utilização do framework TailwindCSS

## Objetivo 

Praticar a utilização do framework TailwindCSS e aplicar praticas de responsividade


## Tecnologias utilizadas

`Html` - construção do esqueleto da página web

`TailwindCSS` - Framework para a estilização responsiva da página web
